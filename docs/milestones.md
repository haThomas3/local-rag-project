# Project Milestones

## Milestone 1 — Local RAG MVP

Goal: Build a minimal local RAG pipeline that can answer questions from local documents.

Deliverables:

- Load local TXT and PDF files
- Split documents into chunks
- Create embeddings
- Store chunks in a local vector database
- Retrieve relevant chunks for a user query
- Generate an answer using an LLM
- Show source references
- Keep API keys and private data out of Git

Out of scope for this milestone:

- Web interface
- User accounts
- Cloud deployment
- Payment system
- Google Drive integration
- YouTube ingestion
